# Musikapplikation

Detta projekt är en webbaserad applikation för att hantera spellistor med låtar. Användaren kan se en lista med låtar, skapa spellistor och se innehållet i dem. Projektet är byggt enligt MVC-arkitektur.

## Funktionalitet

- Lista alla låtar från en `songs.json`-fil
- Välj låtar och skapa spellistor
- Visa en lista med skapade spellistor
- Klicka på en spellista för att visa innehållet
- Låtarna visas i en tabell med kolumner: Titel, Artist och Genre

## Starta applikationen

1. Klona eller ladda ner projektet.
2. Öppna `index.html` i en webbläsare (du kan t.ex. använda VS Code med Live Server).
